board-gui.js – minimal example

This page demonstrates the smallest working setup for the BoardGui widget: just board.css, xiangqi.js and board-gui.js. View the page source for a copy-pasteable starting point. The non-minified versions of the JS and CSS files are accessible if you just removed the ".min" from the URLs, so you can also see the functions.

Setup

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8"/>
  <link rel="stylesheet" href="https://cdn.elephantchess.io/dist/latest/board.min.css">
  <script defer src="https://cdn.elephantchess.io/dist/latest/xiangqi.min.js"></script>
  <script defer src="https://cdn.elephantchess.io/dist/latest/board-gui.min.js"></script>
</head>
<body>

<div id="board-container" class="board-container"></div>

<script>
  window.addEventListener('load', () => {
      const boardGui = new BoardGui(); // defaults to #board-container
      boardGui.loadFen(DEFAULT_START_FEN);
  });
</script>

</body>
</html>
