board-gui.js – minimal example

This page demonstrates the smallest working setup for the BoardGui widget: just board.css, xiangqi.js and board-gui.js. View the page source for a copy-pasteable starting point.

Setup

<link rel="stylesheet" href="https://cdn.elephantchess.io/dist/latest/board.css">
<script defer src="https://cdn.elephantchess.io/dist/latest/xiangqi.js"></script>
<script defer src="https://cdn.elephantchess.io/dist/latest/board-gui.js"></script>

<div id="board-container" class="board-container"></div>

<script>
  window.addEventListener('load', () => {
      const boardGui = new BoardGui(); // defaults to #board-container
      boardGui.loadFen(DEFAULT_START_FEN);
  });
</script>
